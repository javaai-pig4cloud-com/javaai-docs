title: 'Deepseek4j接入GiteeAI R1'
---

本文将介绍如何使用 Deepseek4j 接入 GiteeAI 的 Deepseek-r1:14b 模型。

## GiteeAI 配置准备

### 1. 获取 GiteeAI 访问凭证

1. 登录 [GiteeAI 控制台](https://ai.gitee.com/log4j/dashboard/settings/tokens?action=new)
2. 创建新的 API 密钥并保存

## Deepseek4j 配置

### 1. Maven 依赖配置

在项目的 `pom.xml` 中添加依赖：

```xml
<dependency>
    <groupId>org.deepseek</groupId>
    <artifactId>deepseek4j-spring-boot-starter</artifactId>
    <version>latest-version</version>
</dependency>
```

### 2. 配置文件设置

在 `application.properties` 或 `application.yml` 中添加以下配置：

<Warning>注意模型名称：DeepSeek-R1</Warning>


```properties
deepseek.base-url=https://ai.gitee.com/v1
deepseek.model=DeepSeek-R1
deepseek.api-key=your-gitee-ai-api-key
```

### 3. 代码示例

```java
@Autowired
private DeepSeekClient deepSeekClient;

@GetMapping(value = "/chat", produces = MediaType.TEXT_EVENT_STREAM_VALUE)
public Flux<String> chat(String prompt) {
    return deepSeekClient.chatFluxCompletion(prompt);
}
```
