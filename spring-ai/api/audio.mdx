# Audio Models

Spring AI provides support for various audio processing capabilities through different providers. The audio features are divided into two main categories:

## Transcriptions

Convert speech to text using various AI models:

- [Azure OpenAI Transcriptions](/api/audio/transcriptions/azure-openai-transcriptions)
- [OpenAI Transcriptions](/api/audio/transcriptions/openai-transcriptions)

## Speech Synthesis

Convert text to speech:

- [OpenAI Speech](/api/audio/speech/openai-speech)

Each provider implements specific interfaces for their respective audio processing capabilities, allowing you to work with different AI models while maintaining a consistent API.
