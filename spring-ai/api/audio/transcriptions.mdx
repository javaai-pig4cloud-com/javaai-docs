# Audio Transcriptions

Spring AI provides support for converting speech to text through various AI models. Here are the currently supported implementations:

## Supported Providers

- [Azure OpenAI Transcriptions](/spring4ai/api/audio/transcriptions/azure-openai-transcriptions)
- [OpenAI Transcriptions](/spring4ai/api/audio/transcriptions/openai-transcriptions)

Each provider implements specific interfaces for audio transcription, allowing you to convert speech to text using different AI models while maintaining a consistent API.
