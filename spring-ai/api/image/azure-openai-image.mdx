# Azure OpenAI Image Generation

Spring AI provides integration with Azure OpenAI's DALL-E models for image generation capabilities.

## Configuration

Add the following dependency to your project:

```xml
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-azure-openai-spring-boot-starter</artifactId>
</dependency>
```

## Properties

```properties
spring.ai.azure.openai.api-key=YOUR_API_KEY
spring.ai.azure.openai.endpoint=YOUR_ENDPOINT
spring.ai.azure.openai.image.model=dall-e-3
```

## Usage

```java
@Autowired
private ImageClient imageClient;

// Generate an image
ImageResponse response = imageClient.call(
    new ImagePrompt("A serene landscape with mountains and a lake")
);
```
