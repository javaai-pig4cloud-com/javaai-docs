# QianFan Image Generation

Spring AI provides integration with QianFan's models for image generation capabilities.

## Configuration

Add the following dependency to your project:

```xml
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-qianfan-spring-boot-starter</artifactId>
</dependency>
```

## Properties

```properties
spring.ai.qianfan.api-key=YOUR_API_KEY
spring.ai.qianfan.secret-key=YOUR_SECRET_KEY
spring.ai.qianfan.image.model=stable-diffusion-xl
```

## Usage

```java
@Autowired
private ImageClient imageClient;

// Generate an image
ImageResponse response = imageClient.call(
    new ImagePrompt("A modern Chinese cityscape")
);
```
