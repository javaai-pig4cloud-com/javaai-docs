# ZhiPuAI Image Generation

Spring AI provides integration with ZhiPuAI's models for image generation capabilities.

## Configuration

Add the following dependency to your project:

```xml
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-zhipuai-spring-boot-starter</artifactId>
</dependency>
```

## Properties

```properties
spring.ai.zhipuai.api-key=YOUR_API_KEY
spring.ai.zhipuai.image.model=cogview-3
```

## Usage

```java
@Autowired
private ImageClient imageClient;

// Generate an image
ImageResponse response = imageClient.call(
    new ImagePrompt("A traditional Chinese landscape painting")
);
```
