# Image Models

Spring AI provides support for various image generation models through different providers. Here are the currently supported implementations:

## Supported Providers

- [Azure OpenAI](/api/image/azure-openai-image)
- [OpenAI](/api/image/openai-image)
- [Stability AI](/api/image/stabilityai-image)
- [ZhiPuAI](/api/image/zhipuai-image)
- [QianFan](/api/image/qianfan-image)

Each provider implements the `ImageClient` interface, allowing you to generate images using different AI models while maintaining a consistent API.
