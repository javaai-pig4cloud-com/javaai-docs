# 图像模型

Spring AI 通过不同的提供商提供对各种图像生成模型的支持。以下是当前支持的实现：

## 支持的提供商

- [Azure OpenAI](/api/image/azure-openai-image)
- [OpenAI](/api/image/openai-image)
- [Stability AI](/api/image/stabilityai-image)
- [智谱AI](/api/image/zhipuai-image)
- [千帆](/api/image/qianfan-image)

每个提供商都实现了 `ImageClient` 接口，允许您使用不同的 AI 模型生成图像，同时保持一致的 API。
