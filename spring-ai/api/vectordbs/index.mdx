# Vector Databases

Spring AI provides support for various vector databases to store and retrieve vector embeddings efficiently. These databases are essential for implementing Retrieval Augmented Generation (RAG) patterns and other vector-based operations.

Each integration provides a consistent API while leveraging the unique features of the underlying database. The Spring AI Vector Store API offers a portable interface across all these databases, making it easy to switch between them without changing your application code.

## Common Features

All vector database integrations in Spring AI provide:

- **Vector storage**: Store embeddings generated from your text data
- **Similarity search**: Find the most similar vectors to a query vector
- **Metadata filtering**: Filter results based on metadata attributes
- **RAG integration**: Seamless integration with Spring AI's Retrieval Augmented Generation (RAG) capabilities

For detailed information about specific vector database implementations, please visit the respective documentation pages.
